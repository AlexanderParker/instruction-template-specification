Quick Example

Template:

{
  "$schema": "https://alexanderparker.github.io/instruction-template-specification/schema/v1.0/its-base-schema-v1.json",
  "version": "1.0.0",
  "content": [
    {
      "type": "text",
      "text": "Here are some citrus fruits:\n"
    },
    {
      "type": "placeholder",
      "instructionType": "list",
      "config": {
        "description": "list 5 different citrus fruits",
        "format": "bullet_points"
      }
    }
  ]
}

Compiles to AI prompt:

Here are some citrus fruits:

<<Replace this section with a list. list 5 different citrus fruits. 
Each list item goes on its own line. Use bullet_points to separate each item.>>
